TwitPulse

Sentiment Prediction API

One GET API is provided for predicting sentiment of a tweet give it's status id. This API can be found at the following URL:
/api/twitpulse/sentiment-by-status-id/

The parameters are as follows:

  • statusId: status id of a given tweet
  • queryTerm: (optional) query term that was used to find the given tweet
The sentiment of a tweet is returned by this API, where 0 is negative, 2 is neutral, and 4 is positive.


Example usage: /api/twitpulse/sentiment-by-status-id/?statusId=396418164239511552&queryTerm=Garfunkel+Oates
